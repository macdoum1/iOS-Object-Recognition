S3GetPreSignedURLRequest Class Reference

Inherits from S3Request : AmazonServiceRequest : NSObject
Declared in S3GetPreSignedURLRequest.h

Overview

The GetPreSignedURLRequest contains the parameters used to create a pre signed URL.

Tasks

    Gets and sets the expires property for this request. The time when the signature expires, specified as an NSDate object.

  •   protocol

    Gets and sets the protocol property for this request. Specifies whether the pre signed URL will use http or https. Defaults to https unless otherwise set.
